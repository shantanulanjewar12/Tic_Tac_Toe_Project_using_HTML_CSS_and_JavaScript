# 🎮 Tic Tac Toe Game

A simple and fun Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Features
- Two-player gameplay
- Real-time game status updates
- Highlights the winning combination
- Game reset functionality
- Fully responsive design

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
